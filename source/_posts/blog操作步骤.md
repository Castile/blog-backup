# 创建博客的操作



1. 新建

   ```shell
   hexo n 文章名称
   ```

   此时文章在source/_drafts/ 目录下

2. 发布到post

   ```shell
   hexo p 文章名称
   ```

3. 生成

   ```shell
   hexo g
   ```

4. 部署

   ```shell
   hexo deploy
   ```
